# 🧱 LEGO WeDo 2.0 — Каталог Інструкцій

Інтерактивний, Roblox-подібний веб-каталог для перегляду інструкцій з робототехніки LEGO WeDo 2.0. Створений спеціально для дітей молодшого шкільного віку (1–4 класи), щоб зробити процес вибору та пошуку моделей максимально захопливим.

Цей проєкт є **некомерційним освітнім інструментом**. Усі права на торговельні марки та оригінальні дизайни моделей належать компанії LEGO Group та відповідним авторам.

## 🌟 Особливості
*   **Kids-friendly інтерфейс**: Яскраві кольори, плавні анімації, гігантські тайли моделей.
*   **Глобальний пошук**: Миттєвий пошук за назвою або тегом по всій базі з 380+ моделей.
*   **Система "Обране"**: Локальне збереження улюблених інструкцій з ігровою анімацією "летючого сердечка".
*   **Оптимізовано для планшетів**: Підтримка скролу пальцем/мишкою, великі кнопки, відсутність випадкових виділень тексту чи картинок.

## 🛠️ Технічна реалізація (Важливо!)
Щоб не порушувати ліміти GitHub щодо розміру репозиторію, архітектура проєкту розділена на дві частини:
1.  **Frontend (цей репозиторій)**: HTML, CSS, Vanilla JS та всі оптимізовані зображення-прев'ю. Важить всього кілька мегабайтів. Сайт хоститься через **GitHub Pages**.
2.  **Сховище даних (Google Drive)**: Самі PDF-інструкції (загальним обсягом близько 7 ГБ) зберігаються на Google Drive. У файлі `data.js` зберігаються прямі посилання для попереднього перегляду (`preview`) кожного PDF-файлу.

Завдяки цьому сайт завантажується миттєво, а важкі інструкції підтягуються лише за запитом користувача.
