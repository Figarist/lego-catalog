# 🧱 LEGO WeDo 2.0 — Каталог Інструкцій

Інтерактивний, Roblox-подібний веб-каталог для перегляду інструкцій з робототехніки LEGO WeDo 2.0. Створений спеціально для дітей молодшого шкільного віку (1–4 класи), щоб зробити процес вибору та пошуку моделей максимально захопливим.

Цей проєкт є **некомерційним освітнім інструментом**. Усі права на торговельні марки та оригінальні дизайни моделей належать компанії LEGO Group та відповідним авторам.

## 🌟 Особливості
*   **Kids-friendly інтерфейс**: Яскраві кольори, плавні анімації, гігантські тайли моделей.
*   **🌍 Повна багатомовність (i18n)**: Підтримка трьох мов (**Українська**, **English**, **Русский**) з миттєвим перемиканням без перезавантаження сторінки.
*   **Глобальний пошук**: Миттєвий пошук за назвою або тегом по всій базі з 590+ моделей (працює на обраній мові).
*   **Система "Обране" та "Зібрано"**: Локальне збереження улюблених та виконаних інструкцій. Спеціальний розділ для перегляду власного прогресу.
*   **🎲 Випадкова модель**: Вбудована "рулетка" в стилі Gacha для допомоги у виборі наступного проєкту.
*   **Оптимізовано для мобільних пристроїв**: Повна підтримка планшетів та смартфонів, адаптивний інтерфейс.

## 🛠️ Технічна реалізація
Проєкт побудований на принципах **Offline-First**, чистого коду (Vanilla JS) та високої доступності:
1.  **Модульна архітектура**:
    *   `store.js`: State Manager (LocalStorage, Favorites, Built, I18n state).
    *   `translations.js`: Centralized dictionary for all UI elements and category maps.
    *   `roulette.js`: Standalone Gacha-style animation module.
    *   `app.js`: Core UI Engine, search logic, and i18n switching logic.
2.  **Локалізовані дані**: Файл `data_localized.js` містить повну базу моделей, де назви (`t`) представлені об'єктами для трьох мов.
3.  **AI Context Ready**: У проєкті є файл `AI_CONTEXT.md`, який допомагає ШІ-асистентам миттєво зрозуміти архітектуру для подальшої розробки.
4.  **Data Healing**: Система автоматичного відновлення списку обраного при оновленні посилань у базі даних.
