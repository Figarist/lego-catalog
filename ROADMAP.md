# 🚀 ROADMAP: Майбутні покращення (Offline-First)

Цей документ містить ідеї для майбутнього розвитку проєкту. Оскільки каталог створений для роботи в режимі **Offline-First**, усі описані нижче фічі спроєктовані для роботи виключно на стороні клієнта.

## ✅ Завершено (Completed)
*   **[x] Багатомовність (I18n)**: Реалізовано підтримку UK, EN, RU з миттєвим перемиканням.
*   **[x] Локалізована база даних**: Всі 597 моделей перекладено та адаптовано.
*   **[x] AI Context Architecture**: Створено гайд для ШІ-асистентів для прискорення майбутньої розробки.
*   **[x] Кнопка "Я зібрав це!" (🛠️)**: Користувачі можуть відмічати виконані моделі.
*   **[x] Рулетка (Випадкова модель)**: Додано ігровий елемент вибору інструкції.

## 🎮 1. Гейміфікація (Система досягнень)
Перетворення процесу збірки моделей на гру.
*   **[ ] Локальний прогрес-бар (XP)**: Кожна зібрана модель дає бали досвіду. У шапці сайту з'явиться смужка прогресу.
*   **[ ] Система рівнів**: 3 зібрані моделі — "Новачок", 10 моделей — "Майстер-інженер".
*   **[ ] Бейджі (Ачівки)**: Видача локальних бейджів за серії (наприклад, "Гонщик" за 5 автомобілів).

## 🌍 2. Розширення мов (Localization Expansion)
*   **[ ] Польська мова (PL)**: Наступна мова для додавання в каталог.
*   **[ ] Авто-визначення мови**: Автоматичне перемикання інтерфейсу на мову браузера користувача.

## ⏱️ 3. Speedrun (Таймер збірки)
Заохочення до швидкісної збірки моделей.
*   **Вбудований секундомір**: Кнопка "Почати збірку" в модальному вікні.
*   **Запис рекордів**: Найкращий час зберігається локально для кожної моделі.

## 🔊 4. Ігрові звуки (Sound Design / UI Sounds)
*   **Мікро-звуки**: Короткі звукові ефекти (клік, ачівка, відкриття вікна).
*   **Керування**: Кнопка "Вимкнути звук" у шапці. Всі файли мають бути Royalty Free.

## 👩‍🏫 5. "Режим Вчителя" (QR-завдання)
*   **Генератор посилань**: Вчитель обирає моделі та генерує URL `?lesson=id1,id2`.
*   **QR-код**: Автоматична генерація коду для сканування учнями.
*   **Фільтрований вигляд**: При скануванні каталог показує лише задані вчителем моделі.

---
*Примітка: Усі дані зберігаються виключно в межах одного браузера на одному пристрої (IndexedDB або localStorage).*
